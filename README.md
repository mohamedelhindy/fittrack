# FitTrack 💪

A fitness management web application built using Next.js and Tailwind CSS.

## 🚀 Description

FitTrack helps users:

* Track workouts
* Log exercises and sets
* Monitor progress over time

This project is part of a Database Management Systems course and focuses on designing a system that represents a real-world fitness tracking system.

---

## 🧱 Features

* User authentication (UI only)
* Create and manage workouts
* Add exercises and track sets (reps & weight)
* View workout history
* Track body progress

---

## 🛠️ Tech Stack

* Next.js (App Router)
* React
* Tailwind CSS
* TypeScript
